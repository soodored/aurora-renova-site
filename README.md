# AURORA RENOVA — готовый сайт (шаблон)

## Запуск
Откройте файл `index.html` двойным кликом (работает без сервера).
Если хотите локальный сервер:
- Python: `python3 -m http.server 8080`
- затем откройте `http://localhost:8080`

## Фото
Все изображения лежат в `assets/img/`. Сейчас там **шаблонные** картинки (созданы локально), замените на реальные фото ваших работ с теми же именами файлов.

## Интерактив
- Карточки услуг открывают модальное окно
- Галерея открывается в модальном окне
- Калькулятор ориентировочного бюджета (м² + уровень + доп. работы)
- «Плавающая» кнопка с быстрыми телефонами и копированием номера
